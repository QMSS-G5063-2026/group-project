# Does the AI Trend Follow the Patterns of Earlier YC Trends?

**Columbia University QMSS GR5063 — Data Visualization, Spring 2026**  
Jiahao Huang (jh5140) · Xiang Fan (xf2300) · Zhicheng Jiang (zj2433) · Yehua Huang (yh3932)

An interactive Streamlit app comparing AI with three reference trends in Y Combinator's portfolio (2011–2025): Enterprise SaaS, Fintech, and Developer Tools.

---

## The Argument

Past YC trends were vertical sectors — each with its own market, vocabulary, and geography. AI behaves differently across three independent measurements:

1. **Text** — AI-associated vocabulary has spread into every other sector's company descriptions, and the shift tracks ChatGPT's 2022 launch across the entire portfolio.
2. **Coexistence** — AI grows *inside* other sectors. A significant share of SaaS, Fintech, and DevTools companies are simultaneously tagged AI — a pattern no prior trend shows.
3. **Geography** — AI companies have remained concentrated in the Bay Area while non-AI startups dispersed, suggesting physical proximity to AI talent and infrastructure still matters.

**Requirements:** Python 3.10+

```bash

git clone <repo-url>

# 2. Install dependencies

python prep_data.py

streamlit run app.py
```

## Data

**Source:** [YC Companies dataset by DataHive AI on Hugging Face](https://huggingface.co/datasets/datahiveai/ycombinator-companies)

- 5,080 companies across 31 YC batches, 2011–2025
- Fields used: batch, tags, one-liner, city, country
- Raw CSV: `data/yc-companies.csv` (included, 9.4 MB)
- Cleaned parquet: `data/companies_clean.parquet` (generated by `prep_data.py`, git-ignored)
